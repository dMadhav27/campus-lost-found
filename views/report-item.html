Security Verification

Anti-Fraud Protection

Create security questions that only the true owner would know. This prevents false claims and ensures items reach their rightful owners.

Security Tips
  • Ask about specific details only you would know (contents, hidden damage, personal modifications)
  • Avoid obvious questions like "What color is it?" that anyone can guess from photos
  • Include details about when/where it was purchased or received
  • Ask about specific text, stickers, or markings not visible in photos
  • Minimum 2 questions required, maximum 5 questions allowed
